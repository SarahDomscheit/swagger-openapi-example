# Swagger OpenAPI Lernprojekt

Dieses Repository ist ein **Lernprojekt** zum Kennenlernen und Üben von Swagger/OpenAPI Spezifikationen.

## Über dieses Projekt

In diesem Projekt geht es darum, die Grundlagen von OpenAPI zu verstehen und zu lernen, wie man API-Dokumentationen mit Swagger erstellt und pflegt.

## Was ist OpenAPI/Swagger?

- **OpenAPI** ist eine Spezifikation zur Beschreibung von RESTful APIs
- **Swagger** ist ein Set von Tools zur Arbeit mit OpenAPI-Spezifikationen
- Ermöglicht die standardisierte Dokumentation von APIs

## Inhalt

- [`swagger.yaml`](swagger.yaml) - OpenAPI Spezifikationsdatei

## Visualisierung

Um die API-Dokumentation zu visualisieren, kannst du:

1. [Swagger Editor](https://editor.swagger.io/) im Browser öffnen
2. Den Inhalt der `swagger.yaml` Datei dort einfügen
3. Die interaktive Dokumentation ansehen

## Lernziele

- OpenAPI/Swagger Syntax verstehen
- API-Endpunkte dokumentieren
- Request/Response Schemas definieren
- Best Practices für API-Dokumentation lernen
